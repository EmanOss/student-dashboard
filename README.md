# student-dashboard

A simple dashboard for students to be able to view their courses' announcements and quizzes.

## Features
- user login/registration
- localization (English and Arabic locales available)

## Languages used
### Backend
- Nodejs
- MongoDB
### Frontend
- React (Typescript)
- Material UI Library

## Time in developement
3 days
